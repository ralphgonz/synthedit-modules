Type:  		Waveform

Description:	Generates noise which can be varied from white noise
		(uniform probability distribution) to a sparse, 'static'
		noise (cauchy distribution). The intensity can be
		controlled via an external gain or envelope input.
		See Help button for details.

Setup:		Delete any previous versions and copy to
                SynthEdit\Modules\Waveform
